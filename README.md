# Blog Project - PHP MySQL CRUD Application

## Project Overview
This is a simple Blog Management System built using PHP and MySQL.  
It allows users to register, login, create posts, edit posts, delete posts, and view posts with search and pagination features.

---

## Task 4 - Security Enhancements

## Authentication System
- User Registration implemented
- Secure Login system using sessions
- Password hashing for secure storage

## User Roles
- Admin
- Editor

## Role-Based Access Control
- Admin can create, edit, and delete posts
- Editor has limited permissions

## Security Improvements
- Prepared Statements used to prevent SQL Injection
- Input validation added in forms
- Session management implemented
- Secure authentication flow
